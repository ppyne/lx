# abs

Absolute value

Domain: Numeric and math

---

### Description

`abs(value) : int|float`

Returns the absolute value of `value`.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns a float if `value` is a float, otherwise an int.

### Examples

```php
var_dump(abs(-4.2));
var_dump(abs(5));

/* Will output:
float(4.2)
int(5)
*/
```
