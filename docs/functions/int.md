# int

Cast to int

Domain: Casting helpers

---

### Description

`int(value) : int`

Converts `value` to an integer.
String values support decimal, hex (0x), binary (0b), and octal (leading 0).

### Parameters

- **`value`**: The value to process.

### Return Values

Returns an integer.

### Examples

```php
var_dump(int("42"));

/* Will output:
int(42)
*/
```
