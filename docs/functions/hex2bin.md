# hex2bin

Hex to binary string

Domain: Extensions: hex

---

### Description

`hex2bin(string) : string|undefined`

Converts a hexadecimal string to binary.

### Parameters

- **`string`**: The input string.

### Return Values

Returns the binary string, or `undefined` on invalid input.

### Examples

```php
print(hex2bin("4869") . "\n");

/* Will output:
Hi
*/
```
