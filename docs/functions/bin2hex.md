# bin2hex

Binary to hex string

Domain: Extensions: hex

---

### Description

`bin2hex(string) : string`

Converts a binary string to its lowercase hexadecimal representation.

### Parameters

- **`string`**: The input string.

### Return Values

Returns a string.

### Examples

```php
print(bin2hex("Hi") . "\n");

/* Will output:
4869
*/
```
