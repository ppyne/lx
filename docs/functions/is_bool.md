# is_bool

Boolean type check

Domain: Types and inspection

---

### Description

`is_bool(value) : bool`

Checks whether `value` is a boolean.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_bool(true) . "\n");

/* Will output:
true
*/
```
