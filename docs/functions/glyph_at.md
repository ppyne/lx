# glyph_at

Get UTF-8 glyph at index

Domain: Extensions: utf8

---

### Description

`glyph_at(string, index) : string|undefined`

Returns the UTF-8 glyph (codepoint) at `index` in `string`.
Indexing is 0-based. Returns `undefined` if the index is out of range or
the input is not valid UTF-8.

### Parameters

- **`string`**: The input string.
- **`index`**: The glyph index (0-based).

### Return Values

Returns the glyph as a string, or `undefined`.
