# keys

Array keys

Domain: Arrays

---

### Description

`keys(array) : array`

Returns an array of keys from `array`.

### Parameters

- **`array`**: The array to operate on.

### Return Values

Returns an array.

### Examples

```php
var_dump(keys([10, 20]));

/* Will output:
array(2) {
  [0]=>
  int(0)
  [1]=>
  int(1)
}
*/
```
