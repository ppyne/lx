# deg2rad

Degrees to radians

Domain: Numeric and math

---

### Description

`deg2rad(value) : float`

Converts degrees to radians.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns a float.

### Examples

```php
print(deg2rad(180) . "\n");

/* Will output:
3.14159265358979
*/
```
