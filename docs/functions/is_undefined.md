# is_undefined

Undefined check

Domain: Types and inspection

---

### Description

`is_undefined(value) : bool`

Checks whether `value` is undefined.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_undefined($x) . "\n");

/* Will output:
true
*/
```
