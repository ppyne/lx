# strcmp

Compare strings

Domain: Strings

---

### Description

`strcmp(a, b) : int`

Compares two strings lexicographically.

### Parameters

- **`a`**: The first string.
- **`b`**: The second string.

### Return Values

Returns an integer.

### Examples

```php
print(strcmp("a", "b") . "\n");

/* Will output:
-1
*/
```
