# atan2

Two-argument arctangent

Domain: Numeric and math

---

### Description

`atan2(y, x) : float`

Returns the angle of the vector (`x`, `y`) in radians.

### Parameters

- **`y`**: The y-coordinate.
- **`x`**: The x-coordinate.

### Return Values

Returns a float.

### Examples

```php
print(atan2(1, 1) . "\n");

/* Will output:
0.785398163397448
*/
```
