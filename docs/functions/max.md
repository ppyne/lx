# max

Maximum value

Domain: Numeric and math

---

### Description

`max(a, b) : int|float`

Returns the larger of `a` and `b`.

### Parameters

- **`a`**: The first value.
- **`b`**: The second value.

### Return Values

Returns a float when either argument is a float.

### Examples

```php
print(max(3, 7) . "\n");

/* Will output:
7
*/
```
