# is_int

Integer type check

Domain: Types and inspection

---

### Description

`is_int(value) : bool`

Checks whether `value` is an integer.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_int(1) . "\n");

/* Will output:
true
*/
```
