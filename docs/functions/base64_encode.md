# base64_encode

Base64 encode a string

Domain: Strings

---

### Description

`base64_encode(string) : string`

Encodes `string` using Base64.

### Parameters

- **`string`**: The input string.

### Return Values

Returns the Base64-encoded string.

### Examples

```php
print(base64_encode("Hello") . "\n");

/* Will output:
SGVsbG8=
*/
```
