# log

Natural logarithm

Domain: Numeric and math

---

### Description

`log(value) : float`

Returns the natural logarithm of `value`.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns a float.

### Examples

```php
print(log(2.7182818) . "\n");

/* Will output:
0.999999989530502
*/
```
