# ucfirst

Uppercase first character

Domain: Strings

---

### Description

`ucfirst(string) : string`

Uppercases the first character of `string`.

### Parameters

- **`string`**: The input string.

### Return Values

Returns a string.

### Examples

```php
print(ucfirst("hello world!") . "\n");

/* Will output:
Hello world!
*/
```
