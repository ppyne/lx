# is_array

Array type check

Domain: Types and inspection

---

### Description

`is_array(value) : bool`

Checks whether `value` is an array.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_array([1, 2, 3]) . "\n");

/* Will output:
true
*/
```
