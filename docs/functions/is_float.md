# is_float

Float type check

Domain: Types and inspection

---

### Description

`is_float(value) : bool`

Checks whether `value` is a float.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_float(1.5) . "\n");

/* Will output:
true
*/
```
