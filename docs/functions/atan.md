# atan

Inverse tangent

Domain: Numeric and math

---

### Description

`atan(value) : float`

Returns the arc tangent of `value` in radians.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns a float.

### Examples

```php
print(atan(1) . "\n");

/* Will output:
0.785398163397448
*/
```
