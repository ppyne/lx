# ord

Character code

Domain: Strings

---

### Description

`ord(string) : int`

Returns the character code of the first byte in `string`.
If `string` is empty, the result is 0.

### Parameters

- **`string`**: The input string.

### Return Values

Returns an integer.

### Examples

```php
print(ord("A") . "\n");

/* Will output:
65
*/
```
