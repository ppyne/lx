# is_string

String type check

Domain: Types and inspection

---

### Description

`is_string(value) : bool`

Checks whether `value` is a string.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_string("a") . "\n");

/* Will output:
true
*/
```
