# is_null

Null check

Domain: Types and inspection

---

### Description

`is_null(value) : bool`

Checks whether `value` is null.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns `true` or `false`.

### Examples

```php
print(is_null(null) . "\n");

/* Will output:
true
*/
```
