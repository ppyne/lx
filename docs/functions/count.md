# count

Array size

Domain: Arrays

---

### Description

`count(array) : int`

Returns the number of elements in `array`.

### Parameters

- **`array`**: The array to operate on.

### Return Values

Returns the number of elements, or `0` if the value is not an array.

### Examples

```php
print(count([1, 2, 3]) . "\n");

/* Will output:
3
*/
```
