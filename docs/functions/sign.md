# sign

Sign of number

Domain: Numeric and math

---

### Description

`sign(value) : int`

Returns -1, 0, or 1 depending on the sign of `value`.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns an integer.

### Examples

```php
print(sign(-3) . "\n");

/* Will output:
-1
*/
```
