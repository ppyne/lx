# pow

Power

Domain: Numeric and math

---

### Description

`pow(base, exp) : float`

Returns `base` raised to the power of `exp`.

### Parameters

- **`base`**: The base value.
- **`exp`**: The exponent.

### Return Values

Returns a float.

### Examples

```php
print(pow(2, 3) . LX_EOL);

/* Will output:
8.0
*/
```
