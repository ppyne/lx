# min

Minimum value

Domain: Numeric and math

---

### Description

`min(a, b) : int|float`

Returns the smaller of `a` and `b`.

### Parameters

- **`a`**: The first value.
- **`b`**: The second value.

### Return Values

Returns a float when either argument is a float.

### Examples

```php
print(min(3, 7) . "\n");

/* Will output:
3
*/
```
