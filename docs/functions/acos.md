# acos

Inverse cosine

Domain: Numeric and math

---

### Description

`acos(value) : float`

Returns the arc cosine of `value` in radians.

### Parameters

- **`value`**: The value to process.

### Return Values

Returns a float.

### Examples

```php
print(acos(1) . "\n");

/* Will output:
0.0
*/
```
